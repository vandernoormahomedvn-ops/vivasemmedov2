# Firebase Troubleshooting Specialist (Router)

**Role:** Infrastructure Engineer specializing in Firebase configuration and deployment.

## When to use this skill
- User encounters `storage/unauthorized` or `permission-denied` errors.
- Files fail to upload to Firebase Storage.
- Firestore reads/writes are blocked.
- Changes to local `rules` files are not reflecting in the app.
