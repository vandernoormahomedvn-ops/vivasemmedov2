# Core Pillars & Writing Guidelines

## Core Pillars of Yentelelo Tone
1. **Premium & Sophisticated**: Use elegant language that reflects a high-end service. Avoid slang unless specifically requested for a youthful feature.
2. **Empowering & Visionary**: The copy should make the user feel like they are entering a world of unlimited possibilities ("Explore Sem Fronteiras").
3. **Clear & Direct**: Mobile users skim. Headlines should be punchy (3-5 words), and descriptions should be concise (max 2-3 lines).
4. **Action-Oriented**: Every screen should have a clear "What's next?" through strong Call-to-Actions (CTAs).

## Writing Guidelines

### 1. Headlines (Hooks)
- Focus on the *benefit*, not the feature.
- *Poor*: "Temos muitos vídeos."
- *Premium*: "A Sua Janela para o Entretenimento Infinito."

### 2. Body Text (The Value)
- Create an emotional connection.
- Use active verbs (Descubra, Explore, Conecte-se).
- Align text with the visual depth (Glassmorphism needs "light" and "airy" words).

### 3. CTAs (The Drive)
- Use strong, confident verbs.
- *Standard*: "Próximo", "Ok".
- *Premium*: "Começar Agora", "Explorar Conteúdo", "Criar Minha Experiência".

### 4. Consistency
- Maintain the same "Voice" across the entire app.
- Ensure Portuguese (Mozambique/Portugal) spelling and grammar are impeccable.
