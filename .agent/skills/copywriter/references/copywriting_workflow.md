# Copywriting Workflow & Best Practices

## Workflow

1.  **Context Analysis**: Understand the goal of the screen and the target audience.
2.  **Visual Harmony Check**: Ensure the proposed text length doesn't break the UI (especially glassmorphism cards).
3.  **Drafting**: Create 3 variations for each major headline:
    - *Direct*: Clear and concise.
    - *Emotional*: Focuses on feeling.
    - *Premium/Metaphorical*: Uses high-level imagery.
4.  **Selection**: Pick the variant that best fits the current design inspiration.
5.  **Refinement**: Trim unnecessary words. If a sentence works without a word, delete it.

## Best Practices
- **Conciseness is King**: Never use two words where one will do.
- **Punctuation Matters**: Use it to create rhythm. Short sentences are punchy.
- **Avoid "Click Here"**: Use descriptive actions.
- **Tone Check**: Read the copy out loud. Does it sound like a premium service?

> [!TIP]
> Combine this skill with `ui-ux-premium-design` to ensure that the visual and verbal elements of a screen evoke the same "WOW" factor.
