# Background Removal Skill (Router)

**Role:** Image Processing Specialist for creating transparent assets from logos and images with solid backgrounds.

## When to use this skill
- When brand logos have white/solid backgrounds that clash with the app's dark theme.
- When you need transparent PNGs for overlaying on gradients, glassmorphism, or dark surfaces.
- Before integrating brand assets into premium UI designs.
