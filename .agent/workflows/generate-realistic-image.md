# Generate Realistic Image Workflow

Follow these steps to generate a professional, photorealistic image.

1.  **Define Objective**: Identify the purpose of the image and the core subject.
2.  **Generate Prompt**: Using the `realistic-image-generation` skill, draft a detailed prompt including subject, action, setting, lighting, and technical camera specs.
3.  **Refine Prompt**: Review the prompt for clarity and detail. Ensure it avoids generic adjectives and uses technical photography terminology.
4.  **Execute Generation**: Use the `generate_image` tool with the refined prompt.
5.  **Review & Iterate**: Inspect the generated image for anatomical correctness, lighting consistency, and overall quality. If necessary, adjust the prompt and repeat step 4.
